HTML Introduction

HTML (HyperText Markup Language) is the foundation of web development. It provides the structure and semantic meaning to web content.

What is HTML?

HTML uses tags to define elements like headings, paragraphs, links, images, and more. It's a markup language, not a programming language.

HTML Syntax

Basic Structure

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Page Title</title>
</head>
<body>
    <h1>Hello World</h1>
    <p>This is a paragraph.</p>
</body>
</html>

Core Concepts

Elements and Tags

HTML elements are defined by tags, which are keywords surrounded by angle brackets.

Attributes

Attributes provide additional information about HTML elements.

Semantic HTML

Using meaningful tags like <header>, <nav>, <main>, <article> improves accessibility and SEO.

Examples

Creating a Form

    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    
    <button type="submit">Submit</button>

Common Mistakes

Unclosed Tags

Forgetting to close tags can break your layout.

Missing DOCTYPE

Always include <!DOCTYPE html> at the beginning of your document.

Using Deprecated Tags

Avoid tags like <font> and <center>. Use CSS instead.
